# Security Policy

## Supported Versions

Only the latest major version of WebAnalyzer is actively supported with security updates. We encourage all users to upgrade to the latest release as soon as possible.

| Version | Supported |
| ------- | --------- |
| v3.x    | Yes       |
| < v3.0  | No        |

## Reporting a Vulnerability

We take the security of WebAnalyzer seriously. If you find a security vulnerability, please do not report it publicly in GitHub issues. Instead, please report it directly via email.

Please send vulnerability reports to: **hi@c4softwarestudio.com**

Please include the following information in your report:
* A detailed description of the vulnerability and its potential impact.
* Detailed steps to reproduce the issue (proof-of-concept code, payloads, or screenshots).
* Any potential mitigations or recommendations.

We will acknowledge receipt of your report within 48 hours and work with you to resolve the issue as quickly as possible. Once fixed, we will publish a new release and credit you for the discovery.
